Activity 1: Virtual Art Gallery

Create a Virtual Art Gallery using Pygame that will display a background and a randomly selected artwork from a list and every time the user clicks on the screen, the displayed artwork will change randomly.

Step 1: Download the Activity Template and Open it in Visual Studio Code.

Step 2: Create a new file and name it main.py



Step 3: Import Required Modules
Import the pygame and random modules.


Step 4: Initialize Pygame
Call pygame.init() to initialize all Pygame modules.


Step 5: Create the Game Window

  • Define a variable WIDTH and HEIGHT and set them to 800 and 600 respectively.

  • Create the game window using pygame.display.set_mode() and store it in a variable called screen.

  • Set the window title using pygame.display.set_caption().

Step 6: Load Images

  • Load the background image using pygame.image.load() and store it in a variable bg_img.

  • Create a list ‘images’ that contains five images, each loaded using pygame.image.load("imageX.png"), where X is the image number.

Step 7: Initialize Variables

  • Create a boolean variable running and set it to True.

  • Select a random image from the images list using random.choice(images) and store it in a variable image.

Step 8: Start the Game Loop
Use a while loop that runs as long as running is True.


Step 9: Display the Background Image
Inside the while loop, use screen.blit(bg_img, (0, 0)) to draw the background image at position (0, 0).


Step 10: Display the Selected Image
Inside the while loop, use screen.blit(image, (215, 143)) to draw the selected image at position (215, 143).


Step 11: Handle Events

  • Inside the while loop, iterate through all events fetched using pygame.event.get()

  • Inside for loop, If event.type == pygame.QUIT.
    • Set running = False to exit the loop when the close button is clicked.


  • Inside the for loop, if event.type == pygame.MOUSEBUTTONDOWN:
    • Select a new random image using random.choice(images) and update the image variable.

Step 12: Update the Display
Inside the while loop, call pygame.display.update() to refresh the screen so changes appear.


Step 13: Quit the Game
After exiting the loop, call pygame.quit() to properly close Pygame.

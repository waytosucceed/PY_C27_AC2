Challenge Activity: Measure Accuracy and Give Feedback

Enhance the Virtual Art Gallery by adding two key features:

  1. Timed Delay: The image will only change if at least 5 seconds have passed since the last click. If the user clicks too quickly, a message will remind them to appreciate the artwork before moving on.
  2. No Consecutive Repeats: The newly selected image will always be different from the previous one, ensuring that the same image does not appear twice in a row.

Step 1: Import the time Module


Step 2: Add a Variable to Track Last Click Time
Before the game loop, initialize a variable last_click_time with time.time(). This will store the time of the last mouse click.


Step 3: Store the Initially Selected Image
Before the game loop, store the initially selected image in a variable called previous_image


Step 4: Check Click Timing
Inside the MOUSEBUTTONDOWN event:

  • Get the current time using time.time() and store it in the variable current_time.
  • Check if less than 5 seconds have passed since the last click:
    • Subtract last_click_time from current_time.
    • If the difference is less than 5 seconds: Print "Enjoy the beauty before moving ahead!" in the terminal and do not change the image.

Step 5: Select a New Image and Update Variables.

  • If more than 5 seconds have passed since the last click:
    • Select a new random image and store it in new_image.
    • Ensure the new image is not the same as previous_image. If it is, keep selecting another image until it is different.
    • Once a valid new image is selected:
      • Update previous_image to new_image.
      • Update image to new_image.
      • Update last_click_time to current_time.
